## 🧠 What is KASH.AI?

**KASH.AI** is a personal AI-powered career assistant — built entirely as a single HTML file using the **Gemini API**. It was created by [Kashish Yadav](https://linkedin.com/in/kashish-yadav-6b73782a3), a B.Tech AI & Robotics student, to solve a real problem: *navigating the internship hunt as a student with no connections, no referrals, and limited time.*

Instead of spending hours manually tailoring resumes, decoding JDs, and guessing what recruiters want — KASH.AI automates the entire pre-application intelligence pipeline.

No fancy backend. No cloud infra. Just a browser, an API key, and a file that works.

## 🛠️ Features

### ✅ Currently Live
- 💬 **Conversational Career Assistant** — Ask anything about your resume, career path, skills, or job strategy
- 📋 **JD Analyzer** — Paste a job description; get top 10 ATS keywords, skill gaps, and fit score
- ✍️ **Resume Bullet Rewriter** — Transforms weak bullets into impact-first X-Y-Z formula bullets with metrics
- 🔍 **Profile Roast Mode** — Brutal, honest resume feedback that actually helps
- 🎯 **Role-Specific Generator** — Get tailored resume lines, project highlights, and interview talking points per role

### 🔄 Powered By
- **Google Gemini API** (Gemini 2.5 Pro)
- Pure **HTML + CSS + JavaScript** (zero dependencies, zero build step)
- Runs 100% client-side — your data never leaves your browser

## 🚀 Getting Started

### 1. Clone the repo

```bash
git clone https://github.com/kashishrao68/kash-ai.git
cd kash-ai
```

### 2. Get a Gemini API Key

Head to [Google AI Studio](https://aistudio.google.com/app/apikey) → Create API Key (free tier available)

### 3. Open & Run

```bash
# No installation needed.
# Just open the file in any modern browser:
open index.html
```

### 4. Enter your API key in the UI when prompted — and start talking to your career.
